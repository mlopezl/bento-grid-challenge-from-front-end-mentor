# Frontend Mentor - Solución al Desafío Bento Grid

Esta es mi solución al desafío Bento Grid en Frontend Mentor. El reto me ayudó a mejorar mis habilidades en HTML y CSS al construir un layout de cuadrícula realista y responsive con múltiples tarjetas.

## Resumen
Este proyecto es un layout tipo Bento Grid responsive que consiste en 8 tarjetas. Cada tarjeta contiene una imagen, un título y, en algunos casos, una descripción. El diseño se adapta a diferentes tamaños de pantalla, reorganizando las tarjetas para una visualización óptima en escritorio, tablet y dispositivos móviles.

## El desafío
Los usuarios deberían poder:  
- Ver la cuadrícula óptima según el tamaño de la pantalla de su dispositivo.  
- Observar un arreglo visualmente atractivo de las tarjetas inspirado en el diseño original.  
- Acceder a todo el contenido de manera clara y responsive.

## Mi proceso
- Usé **CSS Grid** para posicionar las tarjetas en un layout responsive.  
- Apliqué **Flexbox** dentro de las tarjetas para alinear imágenes, títulos y texto.  
- Organicé el CSS usando **variables** para colores y espaciado, asegurando consistencia.  
- Implementé **media queries** para hacer el diseño responsive en múltiples tamaños de pantalla.  
- Seguí la metodología **BEM** para la nomenclatura de clases y mantener un CSS limpio y legible.

## Tecnologías usadas
- HTML5  
- CSS3  
- CSS Grid  
- Flexbox  
- Propiedades personalizadas de CSS (variables)

## Lo que aprendí
- Cómo estructurar un layout de cuadrícula responsive usando CSS Grid y Flexbox juntos.  
- Aplicar principios de diseño responsive con media queries para escritorio, tablet y móvil.  
- Usar variables CSS para mantener una paleta de colores y espaciado consistente en todo el proyecto.  
- Organizar un proyecto con carpetas de assets, referencias de diseño y CSS de manera escalable.
